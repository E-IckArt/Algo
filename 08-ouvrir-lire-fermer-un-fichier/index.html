Comment stocker des données sans utiliser MySQL ou autre SGBD ?

Deux types de fichiers

Il existe deux types de fichiers que l'on peut utiliser pour stocker un petit nombre de données simplement afin de tester un programme.
Cela permet d'éviter de demander à l'utilisateur de rentrer les données, mais également la création de variables pour travailler avec des données tests.

  • les fichiers .txt : que l'on peut lire.
  • les fichiers .bin : dont la syntaxe n'est pas compréhensible par l'humain.

On peut créer un fichier .csv, y écrire les données, enregistrer et fermer, puis ouvrir ce fichier avec Excel. Les données seront enregistrées dans une feuille du tableur. Il est possible de modifier les données dans Excel.

On ne peut donc pas utiliser le javascript tel qu'on l'a utilisé jusqu'à présent : côté front-end.
